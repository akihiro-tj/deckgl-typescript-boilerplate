# deck.gl TypeScript Boilerplate

## Setup

```
$ yarn
```

## Develop

```
$ yarn start
```

http://localhost:8000

## Lint

```
$ yarn fix
```

## Build

```
$ yarn build
```


## Update from template repository

```
$ git remote add template git@github.com:akihiro-tj/deckgl-typescript-boilerplate.git
```

```
$ git fetch template
$ git merge template/master --allow-unrelated-histories
```
